Цикл for
for (начальные_условия;  условие_для_выполнения_цикла; изменение_параметров_цикла) {}
<pre><code class="language-cpp">for(int i = 1; i &amp;lt; 10; i++) {}
</code></pre>
Но можно и так
<pre><code class="language-cpp">int i = 1;
for (; i &amp;lt; 10;) {
    std::cout &amp;lt;&amp;lt; i &amp;lt;&amp;lt; &quot; * &quot; &amp;lt;&amp;lt; i &amp;lt;&amp;lt; &quot; = &quot; &amp;lt;&amp;lt; i * i &amp;lt;&amp;lt; std::endl;
    i++;
}
</code></pre>
Этот цикл применим к контейнерам разной природы (массивам, векторам, спискам и т. д.).
Пока рассмотрим его на примере строк. 
Цикл позволяет удобно проитерироваться по символам строки, не используя индексов. 
В этом примере мы считываем строку и печатаем отдельно все символы строки и их ASCII-коды:
<pre><code class="language-cpp">#include &amp;lt;iostream&gt;
#include &amp;lt;string&gt;

int main() {
    std::string line;
    std::getline(std::cin, line);
    for (char symbol : line) {
        std::cout &amp;lt;&amp;lt; symbol &amp;lt;&amp;lt; &quot;\t&quot; &amp;lt;&amp;lt; static_cast&amp;lt;int&gt;(symbol) &amp;lt;&amp;lt; &quot;\n&quot;;
    }
}
</code></pre>
