Да, я могу переписать этот код на C++:
<pre><code class="language-cpp">#include &amp;lt;iostream&gt;
#include &amp;lt;vector&gt;

int main() {
    std::vector&amp;lt;std::string&gt; a = { &quot;0&quot;, &quot;1&quot;, &quot;привет&quot;, &quot;мир&quot; };
    std::cout &amp;lt;&amp;lt; a[2] &amp;lt;&amp;lt; &quot; &quot; &amp;lt;&amp;lt; a[3] &amp;lt;&amp;lt; std::endl;
    return 0;
}
</code></pre>
Здесь я использовал std::vector, чтобы создать массив строк, 
поскольку C++ не поддерживает нетипизированные массивы.
