# Базовые селекторы

|                  |                                                                                                                   |
|------------------|-------------------------------------------------------------------------------------------------------------------|
| `div {}`         | Селектор по тегу<br>Выбирает **все** теги данного типа                                                            |
| `.warning {}`    | Селектор по классу<br>`<p class="warning">...</p>`<br>Для нескольких классов одновременно<br>`.button.primary {}` |
| `#main {}`       | Селектор по ID<br>ID должен быть уникален                                                                         |
| `* {margin: 0;}` | Универсальный селектор<br>Может замедлять рендеринг                                                               |

# Иерархические селекторы (отношения элементов)

|                       |                                                                                          |
|-----------------------|------------------------------------------------------------------------------------------|
| `div p {color: red;}` | Потомок (любой уровень вложенности)<br>Выберет все `<p>` внутри `<div>`, на любом уровне |
| `div > p {}`          | Прямой ребёнок<br>Только `<p>`, которые непосредственно внутри `<div>`                   |
| `h1 + p {}`           | Сосед (находится сразу после)<br>Элемент, который идёт **сразу после** `<h1>`            |
| `h1 ~ p {}`           | Все следующие соседи<br>Все `<p>`, расположенные ниже, но в том же родителе              |

# Атрибутные селекторы - главный раздел

|                                 |                                                                                                                                                            |
|---------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `a[target] {}`                  | Имеет атрибут<br>Выбирает `<a>` с **любой** `target`                                                                                                       |
| `input[type="text"] {}`         | Атрибут равен значению                                                                                                                                     |
| `div[class~="item"] {}`         | Атрибут содержит значение через пробел (token)<br>Сработает на: `<div class="item foo bar"></div>`<br>Но **не** сработает на: `<div class="myitem"></div>` |
| `a[href^="https://"] {}`        | Значение атрибута начинается с строки<br>`^=` - "starts with"                                                                                              |
| `img[src$=".png"] {}`           | Значение атрибута заканчивается строкой                                                                                                                    |
| `a[href*="youtube"] {}`         | Значение содержит подстроку                                                                                                                                |
| <code>p\[lang\|="en"] {}</code> | Специальный селектор языка - \|=<br>Совпадение: `lang="en"` и `lang="en-US"`<br>Но не совпадёт `lang="eng"`                                                |

# Псевдоклассы (динамические состояния)

|                       |                       |
|-----------------------|-----------------------|
| button`:hover` {}     | Наведение             |
| input`:focus` {}      | Фокус                 |
| button`:active` {}    | Активный              |
| li`:first-child` {}   | Первый                |
| li`:last-child ` {}   | Последний ребёнок     |
| li`:nth-child(3)` {}  | N-й ребёнок           |
| li`:nth-child(odd)`   | Нечётные              |
| li`:nth-child(even)`  | Чётные                |
| p`:nth-of-type(2)` {} | Только конкретный тип |

# Псевдоэлементы

|                              |               |
|------------------------------|---------------|
| p`::before` {content: "→ ";} | before        |
| p`::after` {content: "→ ";}  | after         |
| p`::first-line` {}           | Первая строка |
| p`::first-letter` {}         | Первая буква  |

# Комбинирование селекторов

|                                     |                                                                   |
|-------------------------------------|-------------------------------------------------------------------|
| `input[type="email"]:last-child {}` | Поиск `<input type="email">`, который является последним ребёнком |
| `.nav a[href*="profile"] {}`        | Все `<a>` внутри `.nav`, у которых href содержит "profile"        |
| `form button[disabled] {}`          | Все disabled кнопки внутри формы                                  |

# Специфичность CSS - как решается конфликт стилей

Очередность:

1. `!important`
2. inline-style `style=""`
3. `#id`
4. `.class`, псевдоклассы, атрибуты
5. теги
6. `*`

Пример:

| Селектор | Ранг    | Комментарий      |
|----------|---------|------------------|
| `#id`    | Высокий | Побеждает класс  |
| `.item`  | Средний | Побеждает тег    |
| `div`    | Низкий  | Проигрывает всем |

# Как выбрать элемент по аттрибутам (итоги)

| Задача                    | Селектор                    |
|---------------------------|-----------------------------|
| Любой элемент с атрибутом | `[attr]`                    |
| Равен значению            | `[attr="value"]`            |
| Начинается с              | `[attr^="prefix"]`          |
| Заканчивается на          | `[attr$="suffix"]`          |
| Содержит строку           | `[attr*="text"]`            |
| Содержит слово            | `[attr~="word"]`            |
| Совпадает по языку        | <code>\[attr\|="en"]</code> |

# Практические мини-примеры

### Выбрать элементы начинающиеся с "data-"

```css
*[data-*]  /* так нельзя */
```

Правильно:

```css
[data-role] {}
[data-id]   {}
[data-type] {}
```

### Все кнопки "внешние ссылки"

```css
a[href^="http"]:not([href*="mysite.com"]) {}
```
