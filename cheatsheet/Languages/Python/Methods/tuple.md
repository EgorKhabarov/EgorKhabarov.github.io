# tuple

### count
tuple.`count`(value)
Возвращает количество вхождений `value` в кортеже
```pycon
>>> (1, 2, 2, 3).count(2)
2
```
### index
tuple.`index`(value, \[start, \[stop]])
Возвращает индекс первого вхождения `value` в кортеже начиная с `start` и заканчивая `stop`
Вызывает ошибку `ValueError`, если элемент не найден
```pycon
>>> (1, 2, 3).index(2)
1
>>> (1, 2, 3, 2).index(2, 2)
3
```
### + и \_\_add\_\_
`+` и tuple.`__add__`(other)
Возвращает новый кортеж являющийся результатом конкатенации текущего кортежа и `other`
```pycon
>>> (1, 2) + (3, 4)
(1, 2, 3, 4)
```
### * и \_\_mul\_\_ или \_\_rmul\_\_
`*` и tuple.`__mul__`(n)
`*` и tuple.`__rmul__`(n)
Возвращает новый кортеж, состоящий из `n` повторений текущего кортежа
Аналогично `__mul__`, вызывается при `n * tuple`
```pycon
>>> (1, 2) * 3
(1, 2, 1, 2, 1, 2)
>>> 3 * (1, 2)
(1, 2, 1, 2, 1, 2)
```
### == или != и \_\_eq\_\_ или \_\_ne\_\_
`==` и tuple.`__eq__`(other)
`!=` и tuple.`__ne__`(other)
```pycon
>>> (1, 2) == (1, 2)
True
>>> (1, 2) != (3, 4)
True
```
### < или > и \_\_lt\_\_ или \_\_gt\_\_
`<` и tuple.`__lt__`(other)
`>` и tuple.`__gt__`(other)
```pycon
>>> (1, 2) < (1, 3)
True
>>> (1, 3) > (1, 2)
True
```
### <= или >= и \_\_le\_\_ или \_\_ge\_\_
`<=` и tuple.`__le__`(other)
`>=` и tuple.`__ge__`(other)
```pycon
>>> (1, 2) <= (1, 2)
True
>>> (1, 2) >= (1, 1)
True
```
