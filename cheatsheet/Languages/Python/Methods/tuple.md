# tuple

| Метод(ы)                                | Описание                                                                                                                                                                                                     |
|-----------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| tuple`.__add__`(other) `+`              | Возвращает новый кортеж являющийся результатом<br>конкатенации текущего кортежа и `other`<br>`(1, 2) + (3, 4) == (1, 2, 3, 4)`                                                                               |
| tuple`.__mul__`(n) `*`                  | Возвращает новый кортеж, состоящий<br>из `n` повторений текущего кортежа<br>`(1, 2) * 3 == (1, 2, 1, 2, 1, 2)`                                                                                               |
| tuple`.__rmul__`(n) `*`                 | Аналогично `__mul__`, вызывается при `n * tuple`<br>`3 * (1, 2) == (1, 2, 1, 2, 1, 2)`                                                                                                                       |
| tuple`.__getitem__`(index)              | Возвращает элемент по указанному индексу<br>`(1, 2, 3)[1] == 2`                                                                                                                                              |
| tuple`.__contains__`(item) `in`         | Проверяет содержится ли элемент в кортеже<br>`2 in (1, 2, 3)`                                                                                                                                                |
| tuple`.__len__` `len(tuple)`            | Возвращает количество элементов в кортеже<br>`len((1, 2, 3)) == 3`                                                                                                                                           |
| tuple`.__iter__`                        | Возвращает итератор для кортежа<br>`for x in (1, 2, 3): print(x) == "1\n2\n3"`                                                                                                                               |
| tuple`.__eq__`(other) `==`              | Проверяет равны ли текущий кортеж и `other`<br>`((1, 2) == (1, 2))`                                                                                                                                          |
| tuple`.__ne__`(other) `!=`              | Проверяет не равны ли текущий кортеж и `other`<br>`((1, 2) != (3, 4))`                                                                                                                                       |
| tuple`.__lt__`(other) `<`               | Проверяет меньше ли текущий кортеж<br>чем `other` (лексикографический порядок)<br>`(1, 2) < (1, 3)`                                                                                                          |
| tuple`.__le__`(other) `<=`              | Проверяет меньше или равен<br>текущий кортеж по сравнению с `other`<br>`(1, 2) <= (1, 2)`                                                                                                                    |
| tuple`.__gt__`(other) `>`               | Проверяет больше ли текущий кортеж, чем `other`<br>`(1, 3) > (1, 2)`                                                                                                                                         |
| tuple`.__ge__`(other) `>=`              | Проверяет больше или равен текущий<br>кортеж по сравнению с `other`<br>`(1, 2) >= (1, 1)`                                                                                                                    |
| tuple`.__hash__`                        | Возвращает хэш-код кортежа<br>(требуется для использования<br>в качестве ключа в словарях)<br>`hash((1, 2, 3))`                                                                                              |
| tuple`.count`(value)                    | Возвращает количество вхождений `value` в кортеже<br>`(1, 2, 2, 3).count(2) == 2`                                                                                                                            |
| tuple`.index`(value, \[start, \[stop]]) | Возвращает индекс первого вхождения `value` в кортеже<br>начиная с `start` и заканчивая `stop`<br>Вызывает ошибку `ValueError`, если элемент не найден<br>`(1, 2, 3).index(2)`<br>`(1, 2, 3, 2).index(2, 2)` |
