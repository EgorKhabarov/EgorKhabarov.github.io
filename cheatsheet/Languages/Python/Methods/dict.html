Метод Описание
dict.clear() Удаляет все элементы
>>> d = {1: 2, 3: 4}
>>> d.clear()
>>> print(d)
{}
dict.copy() Возвращает поверхностную копию словаря
>>> d = {1: 2, 3: 4}
>>> d.copy()
>>> print(d)
{1: 2, 3: 4}
dict.fromkeys(iterable: Iterable, value=None) Создает словарь из заданной последовательности
>>> print(dict.fromkeys(("key1", "key2", "key3"), 0))
{"key1": 0, "key2": 0, "key3": 0}
dict.get(key) Возвращает значение ключа
>>> print({"model": "Mustang", "year": 1964}.get("model"))
Mustang
>>> print({"model": "Mustang", "year": 1964}.get("color", "red"))
red
dict.pop(key) Удаляет и возвращает элемент с заданным ключом
>>> d = {1: 2, 3: 4}
>>> print(d.pop(3))
4
>>> print(d)
{1: 2}
dict.popitem() Возвращает и удаляет последний элемент из словаря
>>> d = {1: 2, 3: 4}
>>> print(d.popitem())
(3, 4)
>>> print(d)
{1: 2}
dict.setdefault(key, default=None) Возвращает значение по ключу. Если ключ отсутствует он вставляет ключ
>>> d = {}
>>> d.setdefault("k", "d")
"d"
>>> d
{"k": "d"}
dict.update(m, **kwargs) Обновляет словарь указанными парами ключ-значение.
>>> d = {1: 2, 3: 4}
>>> d.update({3: 5, 6: 7})
>>> print(d)
{1: 2, 3: 5, 6: 7}
dict.items() Возвращает вид пары словаря (ключ, значение)
>>> d = {1: 2, 3: 4}
>>> print(d.items())
dict_items([(1, 2), (3, 4)])
dict.keys() Возвращает объект просмотра всех ключей
>>> d = {1: 2, 3: 4}
>>> print(d.keys())
dict_keys([1, 3])
dict.values() Возвращает список всех значений в словаре
>>> d = {1: 2, 3: 4}
>>> print(d.values())
dict_values([2, 4])
