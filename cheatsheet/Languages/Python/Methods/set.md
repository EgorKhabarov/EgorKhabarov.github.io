Оператор > для множеств в Python проверяет, содержит ли первое множество
все элементы второго и при этом имеет хотя бы один дополнительный элемент
```python
>>> {1, 2, 3} < {1, 2, 3, 4}
True
>>> {1, 2, 3} < {1, 3, 4, 5}
False
```

# frozenset

В Python «замороженный» означает, что объект не может быть изменен. Например, рассмотрим `set` и `frozenset`:

```pycon
>>> s = set((1, 2, 3))
>>> s
{1, 2, 3}
>>> s.add(4)
>>> s
{1, 2, 3, 4}
>>> fs = frozenset((1, 2, 3))
>>> fs
frozenset({1, 2, 3})
>>> fs.add(4)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'frozenset' object has no attribute 'add'
```
