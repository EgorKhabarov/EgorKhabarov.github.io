float

as_integer_ratio

float.as_integer_ratio() Возвращает кортеж, представляющий рациональное число, близкое к данному целому числу

pycon
>>> 10.5.as_integer_ratio()
(21, 2)

float.conjugate() Возвращает сопряженное комплексное число

pycon
>>> 10.5.conjugate()
