<p>Запустить сервер</p>
<pre><code>./manage.py runserver
</code></pre>
<p>Создать Django проект</p>
<pre><code class="language-bash">django-admin startproject PROJECTNAME
</code></pre>
<p>Перейти в папку с этим проектом</p>
<pre><code class="language-bash">cd PROJECTNAME
</code></pre>
<p>Создать приложение (App)</p>
<pre><code>./manage.py startapp APPNAME
</code></pre>
<p>Сделать миграцию (изменение в базе данных)</p>
<pre><code>./manage.py makemigrations
./manage.py migrate
</code></pre>
<p>Создать админа</p>
<pre><code>./manage.py createsuperuser
</code></pre>
