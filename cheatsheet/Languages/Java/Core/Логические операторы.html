==

Сравнивает ссылки на объекты, если используется с объектами. Для примитивных типов данных (int, char, boolean, и т. д.) сравнивает значения.

equals

Определён в классе Object, и сравнивает содержимое объектов. Стандартная реализация equals() в классе Object сравнивает ссылки, но многие классы (например, String, Integer, и т. д.) переопределяют этот метод для сравнения значений.

switch case

java
int x = 23;
switch (x) {
    case 1:
        System.out.print ("Переменная равна 1");
        break;
    case 56:
        System.out.print ("Переменная равна 56");
        break;
    default:
        System.out.print("Что-то другое");
}
