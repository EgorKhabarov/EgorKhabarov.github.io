<h3>Ограничения генераков</h3>
<pre><code class="language-java">// Ограничение по типу
public class BoundedGenericsExample {
    public static &lt;T extends Number&gt; void printNumber(T number) {
        System.out.println(number);
    }

    public static void main(String[] args) {
        printNumber(10);         // Integer
        printNumber(10.5);       // Double
        // printNumber(&quot;Hello&quot;); // Ошибка компиляции
    }
}
</code></pre>
