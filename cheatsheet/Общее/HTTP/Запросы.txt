GET
    +-----------------------------------------------------+
    | Получение данных с сервера.                         |
    | Не изменяют состояние сервера и обычно используются |
    | для получения данных, не требующих аутентификации.  |
    +-----------------------------------------------------+
POST
    +----------------------------------------------+
    | Отправки данных на сервер.                   |
    | Используются для создания новых ресурсов или |
    | изменения существующих ресурсов на сервере.  |
    +----------------------------------------------+
PUT
    +-------------------------------------+
    | Обновления ресурса на сервере.      |
    | Заменяют текущий ресурс с указанным |
    | идентификатором на новый ресурс.    |
    +-------------------------------------+
DELETE
    +-------------------------------------------------------+
    | Удаления ресурса с сервера.                           |
    | Удаляют ресурс с указанным идентификатором с сервера. |
    +-------------------------------------------------------+
