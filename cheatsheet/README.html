Keyboard

Settings

Press Ctrl + I to open settings

Search

Press Ctrl + F to open Find in page search To search using regular expressions, write a query in the format /pattern/flags

Press Ctrl + Shift + F to open global search This is a full-text search by both cheat sheet names and their contents

[!TIP] You can refine your full-text search using:

+word - the result must include this word -word - the result must NOT include this word word* - match words starting with this term word~1 - fuzzy match (distance 1) word^2 - boost word importance

Examples: +apple juice*
banana~1 -rotten
apple^2 fresh*
