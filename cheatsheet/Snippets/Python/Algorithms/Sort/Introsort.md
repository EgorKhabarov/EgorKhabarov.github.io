# Introsort

Комбинированный алгоритм сортировки, использует [быструю сортировку](?Snippets/Python/Algorithms/Sort/QuickSort), плюс, при превышении глубины рекурсии некоторой величины
(например, логарифма от числа сортируемых элементов), переключается на [пирамидальную сортировку]()
Стандарт для `.NET`
