# MergeSort (Сортировка слиянием)

Ключевым моментом сортировки слиянием является слияние двух массивов
При слиянии массивы сравниваются поэлементо и меньший элемент записывается в результирующий массив после того, как достигнут конец одного из массивов, в результирующий массив переписывается "хвост" оставшегося массива
Совместно со слиянием двух массивов используется рекурсивное разбиение сортируемого массива на сегменты меньшего размера

```python
def mergesort(arr):
    if len(arr) < 2:
        return arr

    result, mid = [], int(len(arr)//2)

    y = mergesort(arr[:mid])
    z = mergesort(arr[mid:])

    while (len(y) > 0) and (len(z) > 0):
       if y[0] > z[0]:
        result.append(z.pop(0))
       else:
        result.append(y.pop(0))

    return result + y + z

arr: list = [28, 64, 2, 1, 13, 0]
print(mergesort(arr))
# [0, 1, 2, 13, 28, 64]
```
