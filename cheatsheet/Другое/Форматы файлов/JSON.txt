JSON - JavaScript Object Notation
Tекстовый формат обмена данными, который основан на синтаксисе JavaScript.
Он используется для хранения и обмена структурированными данными и часто используется
в веб-приложениях для обмена данными между клиентом и сервером.
Пример
<pre><code class="language-python">{
  &quot;name&quot;: &quot;John Smith&quot;,
  &quot;age&quot;: 30,
  &quot;city&quot;: &quot;New York&quot;
}
</code></pre>
<pre><code class="language-python">import json
config = {&quot;BOT&quot;: {&quot;bot_name&quot;: &quot;&quot;, &quot;token&quot;: &quot;&quot;}}
with open(&quot;config.json&quot;, 'w', encoding='UTF-8') as file:
    file.write(json.dumps(config, indent=4, ensure_ascii=False))
with open(&quot;config.json&quot;, 'r', encoding='UTF-8') as file:
    config = json.load(file)
print(config)
</code></pre>

<pre><code class="language-python">import json
config = [{&quot;name&quot;: &quot;Alice Jones&quot;, &quot;age&quot;: 25, &quot;city&quot;: &quot;Los Angeles&quot;},
          {&quot;name&quot;: &quot;John Smith&quot;, &quot;age&quot;: 30, &quot;city&quot;: &quot;New York&quot;},
          {&quot;name&quot;: &quot;Bob Johnson&quot;, &quot;age&quot;: 40, &quot;city&quot;: &quot;Chicago&quot;}]
with open(&quot;test.jsonl&quot;, 'a', encoding='UTF-8') as file:
    file.writelines([str(line)+&quot;\n&quot; for line in config])
with open(&quot;config.jsonl&quot;, 'r', encoding='UTF-8') as file:
    for line in file:
        data = json.loads(line)
        print(data)
</code></pre>
