Программа, которая читает и исполняет команды

Компилятор
Программа, которая преобразует код на одном языке программирования
в тот что может быть понят целевой машиной
sync async Синхронность - программа выполняет одну задачу за раз
Асинхронность - программа может выполнять несколько задач одновременно
ip Internet Protocol
Уникальный числовой идентификатор устройства в компьютерной сети,

ECS Entity-Component-System архитектура, где сущности это ID,
компоненты содержат только данные,
