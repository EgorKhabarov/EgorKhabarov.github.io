Обозначение / Команда Значение Пример использования
HEAD Текущий коммит git show HEAD – показать текущий коммит
HEAD~1 Один коммит назад git show HEAD~1 – показать предыдущий коммит
HEAD~2 Два коммита назад git diff HEAD~2 HEAD – сравнить с двумя коммитами назад
HEAD^ Родитель текущего коммита (аналог HEAD~1) git checkout HEAD^ – перейти на предыдущий коммит
HEAD^^ Два родителя назад git diff HEAD^^ – сравнить с коммитом два шага назад
HEAD^2 Второй родитель merge-коммита git show HEAD^2 – показать второй родитель merge-коммита
HEAD@{n} Состояние HEAD n перемещений назад (из reflog)
git checkout HEAD@{2}		 git checkout HEAD@{2} – перейти в состояние два перемещения назад
branch-name Последний коммит ветки git log branch-name – показать историю ветки
tag-name Коммит, помеченный тегом git checkout tag-name – перейти на коммит по тегу
commit-hash Прямой указатель на коммит по SHA
a1b2c3d		 git checkout a1b2c3d – перейти на коммит по хэшу
