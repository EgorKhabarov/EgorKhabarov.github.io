| Обозначение / Команда | Значение                                                                  |
|-----------------------|---------------------------------------------------------------------------|
| `HEAD`                | Текущий коммит                                                            |
| `HEAD~1`              | Один коммит назад                                                         |
| `HEAD~2`              | Два коммита назад                                                         |
| `HEAD^`               | Родитель текущего коммита (аналог `HEAD~1`)                               |
| `HEAD^^`              | Два родителя назад                                                        |
| `HEAD^2`              | Второй родитель merge-коммита                                             |
| `HEAD@{n}`            | Состояние HEAD n перемещений назад (из reflog)<br>`git checkout HEAD@{2}` |
| `branch-name`         | Последний коммит ветки                                                    |
| `tag-name`            | Коммит, помеченный тегом                                                  |
| `commit-hash`         | Прямой указатель на коммит по SHA<br>`a1b2c3d`                            |
