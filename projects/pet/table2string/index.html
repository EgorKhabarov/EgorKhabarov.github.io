table2string

Python-библиотека для преобразования таблиц в строку - с удобной поддержкой:
  • Переносов строк внутри ячеек
  • Вложенных таблиц, которые красиво сливаются по границам
  • Выравнивания по горизонтали и вертикали
  • Интеграции эмодзи
  • Поддержки ANSI-цветов, форматирования и ссылок

Subtable

Вы можете включать таблицу внутрь таблицы

from table2string import Table, Themes
subtable = Table(
    name="SubTable",
    table=[("1", "2"), ("3", "4")],
)
table = Table(
    name="Table Name",
    table=[("c1", subtable)],
)
table.print(
    v_align=("-",),
    max_width=(2, 8),
    theme=Themes.thin_thick,
)

Formatting

Текст можно форматировать через ANSI-коды (цвет, ссылки и др.) или написать своё форматирование, например HTML.

Всё форматирование корректно работает с переносами строк и выравниванием

from table2string import Table, Themes, style, link, Color
red_text = style("w\ne", fg=Color.RED)
example_link = link("https://example.com", "link")
text = f"q{red_text}rty{example_link}"
Table([(text,)]).print(
    max_width=(3,),
    theme=Themes.thin_thick,
)
