Running Text Gif Generator

Инструмент на Python для генерации GIF‑анимаций с бегущей строкой текста. Позволяет комбинировать текст, изображения и другие GIF‑фрагменты, задавая их порядок, направление движения, скорость и цветовую конфигурацию - всё упаковано в удобный класс GIF
Please star this project on GitHub! ⭐

Example

Эту анимацию можно сгенерировать с помощью этого простого кода

from gif import GIF
gif = GIF()
gif.add_text_fragment("text")
gif.save("text.gif")

Chrome dino

You can also generate animations from images

from gif import GIF
gif = GIF(columns=22, rows=24)
gif.add_image_fragment(
    image_path="dino.png",
    direction="up",
    duration=100,
    speed=24,
)
gif.save(path="dino.gif")
